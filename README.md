# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.


# 🛒 DigiTools – Digital Product Marketplace

## 📌 Project Overview

**DigiTools** is a modern and responsive digital product marketplace built with React.
Users can browse premium digital tools, add them to a cart, and simulate a checkout experience with real-time updates and notifications.

---

## 🚀 Technologies Used

* ⚛️ React.js
* 🎨 Tailwind CSS
* 🔔 React Hot Toast
* 🧠 React Hooks (useState, Suspense)
* 📦 JSON Data Handling

---

## ✨ Key Features

### 🛍️ 1. Add to Cart with Smart Validation

* Users can add products to the cart
* Prevents duplicate items
* Displays success/error toast notifications

### 🧮 2. Dynamic Cart Management

* Real-time cart updates
* Remove individual items instantly
* Live total price calculation

### 💳 3. Checkout Functionality

* “Proceed to Checkout” clears the cart
* Shows success message using toast
* Cart count updates globally (Navbar synced)

---

## 📷 UI Highlights

* Clean and modern card-based design
* Fully responsive layout
* Smooth hover effects and transitions

---

## 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/digitools.git

# Navigate to project folder
cd digitools

# Install dependencies
npm install

# Run the project
npm run dev
```
